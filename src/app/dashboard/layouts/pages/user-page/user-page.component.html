Bienvenidos a AtmoTrack-én, tu portal innovador para monitorear y comparar datos meteorológicos de tu municipio con lecturas en tiempo real de un sensor local conectado a un Arduino. En nuestra plataforma, hemos integrado la información más precisa y actualizada de AEMET, complementada con lecturas dinámicas de un sensor de temperatura y humedad conectado a un Arduino en tu PC. Este enfoque único te permite no solo acceder a datos meteorológicos relevantes y precisos, sino también experimentar la interacción directa con la tecnología de IoT (Internet de las Cosas) para una experiencia de usuario más inmersiva y educativa.
